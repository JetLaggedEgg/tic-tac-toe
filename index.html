How to play:
There is currently one game mode. Human vs Human. To play, you and your friend must place tiles down in turns. The winner is the first to get three of their tiles in a row, a draw will result in no awarded points and winning awards one. Unless the 'winner goes first' option is off, the winner will go first the next round.


About this game:
I created this game so that I could enhance my front end web developing knowledge. It's made using HTML and CSS as it's UI and JavaScript for the inner workings and to add functionality to the UI.
